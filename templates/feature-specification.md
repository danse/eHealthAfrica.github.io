The following is the skeleton of a Feature Specification template that is generated for each project.

---

### Outline

- Vision
    - Access Details
    - Application
- Use Cases
- Strategic fit
- Goals and Metrics for Success
- Team
- Timeline
    - Proposed Milestones
- [User Roles](#user-roles)
- Survey vocabulary
- Feature Spec
- Non-functional Requirements
    - Documentation
    - Client-side browser compatibility
    - Localization
    - Multi-instance configurability
    - Security
    - Performance
    - Usability
    - Accessibility
    - Compliance
    - Fault Tolerance
    - Extensibility / Upgradeability
    - Maintainability / Testability
- Research
    - Interviews
    - User Tests
    - Photos of the context in which this will be used
- Technical Docs
- Technical Limitations
- Budget
- Questions & Issues
